# PlaymoodTV - Android TV App

This is an [Expo](https://expo.dev) project created with [`create-expo-app`](https://www.npmjs.com/package/create-expo-app) for Android TV.

## Get started

1.  **Install dependencies**

    ```bash
    npm install
    ```

2.  **Start the app**

    ```bash
    npx expo start
    ```

    In the output, you'll find options to open the app in a development build or an Android emulator.

## Project Structure

This project uses `react-navigation` for stack-based navigation. The main entry point is `App.js`, which sets up the navigation container and screen stacks. All screens and components are located in the `src/` directory.
